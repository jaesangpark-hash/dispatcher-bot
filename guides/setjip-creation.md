# **インデクス**

### **設定集とは❓**

登場人物や作品特有の単語などをまとめておく指針書にあたるものです。設定集は作品の１～20話を読みながら作成してください。設定集に基づいて翻訳を進めながら新しい登場人物や単語を追加する形になります。

設定集はクライアントおよび原作の版元の者たちが確認するので丁寧に作成してください。

翻訳実力だけではなく設定集作成も評価基準の一つになります。

✔️　設定集の作成方法

```
１～20話を読む → エディターで作品情報を記入 → 提出
```

✔️設定集の管理方法

```jsx
設定集作成 -> チェッカー確認 -> （修正）-> PM確認 -> （修正）-> クライアント確認 -> （修正）
-> チェック完了 -> 翻訳作業開始 
```

翻訳作業中、翻訳エディターの右側にある参考タブにFIX設定集がアップされます。

（翻訳開始以後1週~1か月後）

- 「FIX」表記があるファイルがFIX設定集です。

![Untitled](Untitled.png)

### 【注意点】

- **1-3話翻訳開始の前に必ずPM&クライアントから修正内容を反映してください**。
- 翻訳の際に発生した内容変更はFIX設定集には反映されません。
    - 翻訳の際に追加されるキャラ＆用語はFIX設定集に記載不要。
- FIX設定集と設定集エディターを比べて一致しない内容はないか確かめる。
- **キャラ名、一人称、単語のフリガナなどはクライアントおよび担当者の許可なしでは変更できませんので変更の前にかならずお問い合わせください。（呼び方などはFIX設定集と異なって構いません）**

- 情報を入れる場所：原作を読みながら入れる情報があったら記入してください。

## 4. 用語

![Untitled](Untitled%208.png)

1. 作中登場する固有名詞などの翻訳の際参考になれるものはすべて入れてください。
    1. 用語にフリガナをつけてください。
        1. **一般的に読まれる簡単な単語については記載不要**
        2. **通常と異なる読み方をするものや、小中学生には読みにくい単語については記載必要**
        3. 地名や固有名詞にはルビ必要
2. 「用語/場所を追加」から用語を追加してください。
3. カテゴリー :  用語に当たる言葉なのか、場所に当たる言葉なのかチェックしてください。
4. 初登場話数 : 初めて登場する話数を入れてください（空欄は絶対禁止）
5. 単語 : 左側→原語（中国語） / 右側 →訳語（日本語）
6. 画像 : 必須ではありませんが、入れる必要のある用語なら添付してください。
7. 説明 : 左側（中国語）は記入する必要ありませんが、右側は簡単にでも構わないので必ず記入してください。

```jsx
用語集についての補足説明（241127追加）

用語集には基本的に該当作品にだけ出てくる固有名詞を記載するのが原則
簡単な用語でも原文に当てはまる訳語が複数のあり、
翻訳をするときに統一する必要がある場合は記載必要。
しかし、固有名詞以外の簡単な単語は記載不要
Ex)法学部、資料など
Ex)眼镜 -> 眼鏡（メガネ）のように簡単な単語が一般的な意味でのみ訳される場合は記載不要
```

## 6．提出方法

# 設定集管理方法及び注意事項

## 名前（フリガナ含め）、一人称、用語名は作品担当者及びクライアント確認を得ずには変更不可能

- キャラ追加、用語追加、呼び方の変更は翻訳しながら更新できる範囲ですが、キャラ名、一人称は必ずクライアントから更新した設定集（FIX設定集）に合わせて翻訳してください。
    - 呼び方の場合、ニュアンスが完全に変わるほどの修正はお問い合わせ必要
- もし、変更が必要な場合は必ず作品担当者に問い合わせください。
